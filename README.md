# hexo-deployer-tcb


## Install
```$xslt
npm install hexo-deployer-tcb --save
```

## Usage
You can configure this plugin in _config.yml.
```$xslt
# You can use this:
deploy:
  type: tcb
  secretId: < # your envId >
  secretKey: < # your secretKey >
  envId: < # your envId >
```
You can get it from here.[Tencent Cloud Console](https://console.cloud.tencent.com/cam/capi)
